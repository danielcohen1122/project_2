# TriScreen — סרטון פרסומת

`triscreen-promo.mp4` — סרטון פרסומת אנכי (9:16, 1080×1920, ~30 שניות, H.264)
המיועד לטיקטוק, אינסטגרם ריל/סטורי ופייסבוק.

## מבנה הסרטון (6 סצנות)
1. וו פתיחה — "עדיין עובד על מסך אחד?"
2. חשיפה — אנימציית שלושת המסכים + "שלושה מסכים. אפשרויות אינסופיות."
3. מספרים — 15.6" · 1080P · 10 שניות הרכבה · 3× שטח עבודה
4. יתרונות — USB-C יחיד · 1.3 ק"ג · 30 יום החזר
5. קהלי יעד — מפתחים · סוחרים · היי-טק · עורכי תוכן
6. מחיר + CTA — 1,099 ₪ כולל משלוח חינם + הזמנה/וואטסאפ

## איך מפיקים מחדש
המקור הוא `video.html` (אנימציית HTML/CSS מונעת JS).
מקליטים אותו ל-MP4 עם Playwright + ffmpeg:

1. `npm i playwright` והקלטת הדף ב-viewport 1080×1920 עם `recordVideo`.
2. המרת ה-webm ל-MP4:
   `ffmpeg -i out.webm -vf "fps=30,format=yuv420p" -c:v libx264 -crf 20 -movflags +faststart triscreen-promo.mp4`

הסרטון ללא סאונד בכוונה — מומלץ להוסיף מוזיקה טרנדית ישירות בפלטפורמה.
